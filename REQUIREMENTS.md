# Audiobookshelf iOS 客户端需求说明书

## 1. 项目概述
本项目旨在开发一款基于 iOS 17.0+ 平台的 Audiobookshelf (ABS) 原生客户端。该应用将允许用户连接到自托管的 Audiobookshelf 服务器，进行有声书和播客的流媒体播放及离线收听。

## 2. 目标用户
- 自托管 Audiobookshelf 服务器的用户。
- 需要在移动设备（iPhone）上收听有声书和播客的用户。
- 追求原生 iOS 体验和离线播放功能的用户。

## 3. 运行环境
- **操作系统**: iOS 17.0 及以上。
- **设备**: iPhone (未来可扩展至 iPad/CarPlay)。
- **网络**: 需要网络连接进行数据同步和流媒体播放，支持离线模式。

## 4. 功能需求 (Functional Requirements)

### 4.1 认证与连接 (Authentication)
- **服务器连接**: 用户输入服务器地址（支持 HTTP/HTTPS）。
- **用户登录**: 支持用户名/密码登录。
- **多用户支持**: 支持快速切换用户（可选）。
- **Token 管理**: 安全存储 Session Token (Keychain)。

### 4.2 媒体库 (Library)
- **展示**: 以网格 (Grid) 或列表 (List) 形式展示有声书和播客。
- **筛选与排序**:
  - 按作者、系列、添加时间、进度排序。
  - 筛选“未读”、“进行中”、“已读”。
- **搜索**: 全局搜索书籍、作者、播客、剧集。
- **详情页**: 展示封面、元数据（作者、简介、时长）、章节列表、系列信息。

### 4.3 播放器 (Player)
- **流媒体播放**: 支持 MP3, M4B, FLAC 等常见音频格式的流式播放。
- **播放控制**: 播放/暂停、快进/快退（自定义时间）、上一首/下一首（章节）。
- **进度同步**: 实时或定期向服务器同步播放进度。
- **倍速播放**: 支持 0.5x - 3.0x 播放速度调节。
- **睡眠定时器**: 支持按时间（15m, 30m...）或“本章结束”设置定时停止。
- **章节导航**: 支持查看和跳转章节。
- **后台播放**: 支持应用后台运行时的音频播放及锁屏控制 (MPNowPlayingInfoCenter)。

### 4.4 离线下载 (Downloads)
- **单集/整书下载**: 支持下载音频文件到本地。
- **离线播放**: 优先播放本地文件，无网络时自动切换离线模式。
- **下载管理**: 查看下载进度、暂停/恢复下载、删除已下载文件以释放空间。

### 4.5 iOS 特性集成 (iOS 17 Features)
- **Live Activities**: 在锁屏和灵动岛 (Dynamic Island) 显示当前播放进度和控制。
- **Widget**: 桌面小组件，快速进入最近播放。

## 5. 非功能需求 (Non-functional Requirements)
- **性能**: 列表滚动流畅，封面加载迅速（需实现图片缓存）。
- **UI/UX**: 遵循 Apple Human Interface Guidelines，适配深色模式 (Dark Mode)。
- **电池优化**: 优化后台播放和网络请求，减少电量消耗。
- **兼容性**: 严格适配 iOS 17 API (如 Observation 框架, SwiftData/CoreData)。

## 6. 数据模型参考
参考 `audiobookshelf_src` 中的数据结构：
- **User**: 用户权限、设置。
- **LibraryItem**: 书籍/播客实体。
- **AudioTrack**: 音频文件信息。
- **PlaybackSession**: 播放会话与进度。
